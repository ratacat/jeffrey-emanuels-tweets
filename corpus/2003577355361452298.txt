Give the gift of slop creation to your loved ones this holiday season.

But seriously, if you have the means, and you know someone who is struggling and doesn’t know what to do next and is understandably apprehensive about the future, try asking them if they’re interested in learning how to make software using AI agents.

If they’re actually motivated, you can create a limited-use credit card number and have them use it to sign up for Claude Max and GPT Pro and rent a cloud server and see how much they can get done in a month. At the very least, they should learn something for the ~$500 or so.

And if someone is really driven and diligent, you might be shocked how much they can do once they cross the initial (admittedly steep) hump of getting a cloud server set up with all these agents. 

It’s the closest thing there is to having real magical powers. And it’s a lot of fun, too.
